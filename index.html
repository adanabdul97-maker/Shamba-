📊 Current Data Status
⬇️ Create a Backup

This saves everything — all crops, activities, seasons, and records — into one backup file (.json). Save it to iCloud Drive or Google Drive so it's always safe.

📱 iPhone instructions after tapping Backup:
1. A file download will start
2. Tap "Save to Files" when prompted
3. Choose iCloud Drive → create a folder called "Shamba Backups"
4. Tap Save — done! ✅
Do this weekly or after every major data entry session.
⬆️ Restore from Backup

Use this if you ever lose your data or switch to a new phone/browser. Select the .json backup file you saved earlier and all your data will be restored instantly.

⚠️ Important: Restoring will replace all current data with the backup. Make a fresh backup first if you have new entries you want to keep.
⏰ Backup Reminder

The app will remind you to back up if you haven't done so in 7 days. You can also manually check your last backup date above.

✅ Best practice routine:
• After entering a week's worth of activities → Backup
• At the end of every season → Backup
• Before clearing browser history → Backup first!
• When switching phones → Backup on old phone, Restore on new
