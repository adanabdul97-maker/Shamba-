⬇️ Create a Backup

Saves everything — all crops, planting records, activities, staff, seasons — into one file. Save to iCloud Drive or Google Drive.

📱 iPhone steps: Tap Download → tap "Save to Files" → choose iCloud Drive → Shamba Backups → Save ✅
⬆️ Restore from Backup

Select your .json backup file to restore all data instantly.

⚠️ Warning: Restoring replaces ALL current data. Backup first if you have new entries.
